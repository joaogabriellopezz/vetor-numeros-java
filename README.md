# Vetor de Números

## Descrição do programa

O programa recebe 5 números inteiros do usuário através de caixas de diálogo (JOptionPane). Ele processa esses valores em uma classe de serviço separada para identificar o maior número par, o menor número ímpar, além de calcular o somatório e a média de todos os valores.

## Como executar o programa

1. Certifique-se de ter o JDK instalado na máquina.
2. Abra o projeto na sua IDE (Eclipse, VS Code ou IntelliJ).
3. Execute a classe `ClassePrincipal.java`.
4. Digite um número inteiro em cada uma das 5 caixas de entrada que aparecerem.

## Exemplo de entrada e saída

* **Entrada (Digitada no JOptionPane):** `2`, `5`, `8`, `11`, `4`
* **Saída (Exibida na tela):** 
  * Maior número par: 8
  * Menor número ímpar: 5
  * Somatório de todos os valores: 30
  * Média de todos os valores: 6.0
